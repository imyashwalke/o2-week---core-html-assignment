  1. Div (div) =
    Div creates a container or a division within a web page that can be used to group other html elements such as text, img, forms, and other html tags into a logical section or a block. the (div) tag has semanting meanings or a function, but it allows developers to apply style, script or other attributes to a group of a elements as a unit.

Self-closing tags

in htm, some elements are self closing tags or void elements because they dont require a separate closing tags. these elements are used for elements that don't contain any content or don't have a closing counterpart. ex-img, input, br, meta.

Meta tag =

meta tag in html is used to provide information about a web page to search engines, browsers, and other web services.they are placed within the head section of a html document and do not affect the apperance of the web page.

Examples of meta tags:
  1. Description
    • this meta tag provide a brief description of the web page content, which may be displayed in search engine result.
  2. Author
    • this meta tag provides the name of the Author who created the web page.

Q10. What are global attributes? List any 5 global attributes.

Global attributes are attributes common to all HTML elements; they can be used on all elements, though they may have no effect on some elements. Global attributes may be specified on all HTML elements, even those not specified in the standard.

list of 5 global attributes
  1. style
  2. title
  3. id
  4. lang
  5. accesskey
