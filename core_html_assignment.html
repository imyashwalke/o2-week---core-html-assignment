Q9. Create an ordered list of the most common text formatting tags in HTML. Within each list item, use an unordered list to list the specific use cases and best practices for that tag.

  1. Bold (b)
    • The b tag specifies bold text without any extra importance.
    • Defines bold text
  2. emphasis (em)
    • Defines emphasized text
    • use to indicate emphasis or to differentiate between words or phrases
  3. strong
    • The strong tag is used to define text with strong importance.
    • The content inside is typically displayed in bold.
  4. Strikethrough (s)
    • The s tag specifies text that is no longer correct, accurate or relevant.
    • used to define deleted text in a document, use the del tag for that.
